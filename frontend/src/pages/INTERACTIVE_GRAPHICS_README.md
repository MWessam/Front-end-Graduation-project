# Interactive Graphics Development Guide

## Library: React Konva

We're using **React Konva** for building Brilliant-style interactive graphics. It's a React wrapper for Konva.js, a powerful 2D canvas library.

### Why React Konva?

- ✅ **React-friendly**: Works seamlessly with React components
- ✅ **Interactive**: Built-in drag, drop, click, and hover events
- ✅ **Performant**: Uses Canvas API for smooth animations
- ✅ **Flexible**: Perfect for math visualizations, graphs, geometric shapes
- ✅ **Similar to Brilliant**: Canvas-based like Brilliant's visualizations

### Alternative Libraries (if needed later)

1. **Mafs** - Specifically for math visualizations (React components)
2. **D3.js** - Powerful but more complex, better for data visualization
3. **Fabric.js** - Another canvas library option
4. **Paper.js** - Vector graphics library

### Getting Started

The test page is at `/` (root route) and also available at `/test-graphics`.

### Basic Concepts

- **Stage**: The main container (like a canvas)
- **Layer**: Groups of shapes (like layers in design software)
- **Shapes**: Circle, Line, Rect, Text, Group, etc.
- **Events**: onClick, onDrag, onHover, etc.

### Example Use Cases for Math

1. **Graph Plotting**: Interactive function graphs
2. **Geometry**: Draggable shapes, angles, triangles
3. **Coordinate Systems**: Points, vectors, transformations
4. **Calculus**: Area under curves, derivatives visualization
5. **Statistics**: Interactive charts and data visualization
