# Pure Project - React Conversion

This is the React conversion of the Pure HTML/CSS/JS project with React Router for transitions.

## Project Structure

- `src/pages/` - Contains all page components
- `src/App.jsx` - Main app component with React Router setup
- `src/main.jsx` - Entry point

## Pages

1. **Achievements** - `/achievements`
2. **AllCourses** - `/all-courses`
3. **Assignment** - `/assignment`
4. **Concept** - `/concept`
5. **ConceptLesson** - `/concept-lesson`
6. **Courses** - `/courses`
7. **Edit** - `/edit`
8. **Essay** - `/essay`
9. **EssayResult** - `/essay-result`
10. **Instructions** - `/instructions`
11. **Lectures** - `/lectures`
